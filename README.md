# 勤怠管理システム（attendance）

Spring Boot を用いて開発した、管理者・社員権限対応の勤怠管理 Web アプリケーションです。

---

## システム概要

本システムは、企業における勤怠データの正確性と保全性を重視して設計した業務向け Web アプリケーションです。

- 社員打刻機能（出勤・退勤・休憩）
- 残業時間自動計算
- 週 40 時間チェック
- 修正申請フロー
- 管理者承認機能
- CSV 出力対応
- 社員番号ユニーク管理
- ロールベース認可（ADMIN / EMPLOYEE）

---

## 技術構成

- Java 17
- Spring Boot 3.x
- Spring Security
- PostgreSQL
- Thymeleaf
- JPA (Hibernate)

---

## 設計思想（データ保全重視）

本システムは単なる打刻アプリではなく、「データ改ざん防止」と「整合性維持」を重視しています。

### 1. データベースレベルでの整合性担保

- UNIQUE 制約（社員番号）
- 外部キー制約（attendance → users）
- NOT NULL 制約
- 不正な労働時間を出さないロジック制御

### 2. 修正申請フローによる改ざん防止

- 打刻データの直接編集禁止
- 修正は申請制
- 管理者承認後に反映

### 3. 権限制御

- Spring Security によるロール制御
- 管理者ページへのアクセス制限

## 主な機能

| 機能         | 内容                   |
| ------------ | ---------------------- |
| 打刻機能     | 出勤・退勤・休憩登録   |
| 残業計算     | 8 時間超過分を自動算出 |
| 週次チェック | 40 時間超過確認        |
| 修正申請     | 社員から管理者へ申請   |
| CSV 出力     | 管理者用データ出力     |
| ユーザー管理 | 管理者による登録・編集 |
